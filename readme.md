Markdown syntax supports the following:

- Headings
- Text Styles (bold, italics, etc.)
- Code Blocks
- Lists (ordered, and unordered)
- Links
- Images
- And more

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

<!-- This text is italic -->

_This text is italic_

<!-- Stront text -->

**This is a strong text**

<!-- Strike Through text -->

~~This text ~~ is strikethrough

<!-- Block Quote -->

> This is a quote

<!-- Links -->
